This application uses Jellyfin login credentials to access a Jellyfin server instance for the purpose of modifying MediaSegment API data.

Credentials and server address are provided by the user and stored as encrypted values for the sole purpose of connecting to the Jellyfin server instance.

Any content beyond the standard settings and dialogs is provided by the Jellyfin server instance and is not retained beyond standard image caching methods.

No personal information (excluding server url, username / password, and API key) is requested by the application or is retained by the application.
